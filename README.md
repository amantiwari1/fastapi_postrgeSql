Install docker in Amazon linux

```
sudo yum install docker -y
sudo systemclt start docker
sudo systemclt enable docker
```
Run docker

```
sudo docker pull amamtiwari1/collegeapi
sudo docker run -d -p 80:5000 amantiwari1/collegeapi
```

